## 🚀 **Быстрый запуск**

### Простой способ:
```bash
python main.py
```

### Professional способ:
```bash
make dev              # Разработка
make prod            # Продакшен
make docker-dev      # Docker разработка
make docker-prod     # Docker продакшен
```

### Docker Compose:
```bash
docker-compose up -d  # Запуск всей инфраструктуры
```

## 📊 **Мониторинг и управление**

### Просмотр логов:
```bash
tail -f bot.log                    # Основные логи
tail -f logs/bot_$(date +%Y%m%d).log  # Файловые логи
```

### Управление процессом:
```bash
ps aux | grep "python main.py"    # Статус
pkill -f "python main.py"         # Остановка
```

### База данных:
```bash
sqlite3 app.db                    # Подключение к БД
.tables                          # Список таблиц
SELECT * FROM applications;      # Просмотр заявок
```
